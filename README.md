# Noughts-and-Crosses-Game
Nought and Cross is a famous game played in notebooks and is a game in which one player draws Xs and another player draws Os inside a set of nine squares, and each player tries to be the first to fill a row of squares with either Xs or Os. This app allows you to have the experience of playing it with an NPC programmed to not make you win.
